# NLW05-NodeJS-Socket

1. Instale o ***Yarn***

``npm install --global yarn``

2. Instale as dependências

``yarn``

3. Execute as ***migrations***

``yarn typeorm migration:run``

4. Teste a aplicação

``yarn dev``
